Apple Shortcuts Setup
One Shortcut per bank — do this once after deploying the worker:
  1. Open Shortcuts app → Automation → +
  2. Choose Personal Automation → Message
  3. Set sender to your bank (e.g. HDFCBK, SBIINB, ICICIB)
  4. Turn off "Ask Before Running"
  5. Add action: Get Contents of URL
  6. Method: POST · URL: Configure worker URL first
  7. Request Body → JSON → Key: sms → Value: Shortcut Input
